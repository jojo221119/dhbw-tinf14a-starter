# General
This is the WWW component of the starter project. 
It is a slightly customized starter project for Angular2 and Webpack.

# Installation
	npm install

# Building
    npm run build:dev

# Running dev
    npm run server:dev

# Testing
## Angular 2 component tests (with karma)
	npm run test
		
## End-2-End testing
	npm run e2e
	
# Usage
`http://starter/health`
